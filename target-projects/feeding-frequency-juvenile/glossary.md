# Glossary — 2026 Feeding Frequency Juvenile

Domain vocabulary for the juvenile feeding-frequency trial. Add new terms here when they first resolve.

## Trial cohorts (mirror the 2026 Juvenile gut evac sibling project)

**Cohort A** — juveniles 10–45 g, fed Biomar 3 mm pellet.

**Cohort B** — juveniles 45–80 g, fed Biomar 4.5 mm pellet.

**Cohort C** — juveniles 80–150 g, fed Biomar 4.5 mm or 6.5 mm pellet (size selection per fish size).

## Methods anchors

**Feeding frequency** — number of discrete meals per day, the trial's primary independent variable. The values chosen will be informed by per-cohort GER kinetics from the sibling Juvenile gut evac project (`~/ResearchProjects/2026 Juvenile gut evac/`).

**GER** — gastric evacuation rate. Inter-meal spacing should be ≥ time-to-target-residual from the cohort's GER curve so each meal starts on an empty stomach. Sibling project produces these curves; this trial uses them.

**Bilbul fish census** — pre-trial census of available juvenile cages at the Bilbul site. Drives cage allocation. Recorded in `Proposal/Bilbul_fish_census_2026-05-04.xlsx`.
