# Glossary — 2023 Open Ponds Whitton

Domain vocabulary for this project. Add new terms here when they first resolve.

**Open pond / free range** — Production configuration where fish swim freely in a pond, no in-pond cage. Lower capex, larger swim volume.

**Redox** — Oxidation-reduction potential of pond water/sediment — a primary water-quality indicator the trial tracks closely (open-pond stocking changes the redox profile vs. caged stocking).

**LM edit** — Lizzie Moore's annotated copy of the trial-design sheet (`Data/DS_OP trial design_18_12 - LM edit.xlsx`).

**Mercatus weights** — Stocking and weight records pulled from MCA's production system (Mercatus) for cage-selection scoring.

**Movement (fish moved to open ponds)** — Discrete event 23 Jan 2024 — cohort transferred from in-pond cages to open-pond configuration. Marks trial start.

**WP9 / WP10** — Whitton pond identifiers (same as the sibling Whitton Feed Trial).
