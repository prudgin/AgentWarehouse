# Glossary — 2025 RAS Feed Biomar

Domain vocabulary for this project. Add new terms here when they first resolve.

**Trial 2.0** — The 2025 iteration; the 2023 RAS Feed Biomar project was Trial 1.0. The numbering tracks Biomar's formulation evolution.

**T0 / T1 / T2 / T3** — Trial timepoints. T0 = stocking (25 Mar 2025); T1 = first intermediate sample (Jul 2025); T2 = second sample (Oct–Nov 2025, includes a small-tanks cull); T3 = final harvest (Feb 2026).

**Finisher feed** — Late-stage diet variant tested as stage 2 / stage 3 sub-investigation under `Data/RAS trial 3 - finisher feeds/`.

**Rod Remke / Lizzie Moore** — Project owner / technical owner respectively. Rod is the Bilbul RAS site manager (`farm 1444D Bilbul Road`).

**Pellet weight soak test** — Pre-trial assessment of pellet swelling/sinking after immersion. Done 2–3 Apr 2025 (`Data/Pellet weight soak test 2.4.25 - 3.4.25.xlsx`).

**Feed name to colour conversion** — Mapping used at the tank level so operators can identify the feed treatment by pellet colour without leaking the brand identity (`Reports/feed name to colour conversion.pptx`).

**BiomarMCA Trial 2025** — Joint MCA-Biomar trial brand; the processing+analysis report uses this label (`Reports/Fish processing and analysis_BiomarMCA Trial 2025.pdf`).
