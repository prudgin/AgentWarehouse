# ADRs — 2026 Stanbridge gut evac

Methodology decisions. The 3-of-3 admission test applies: an ADR is warranted when a decision is hard to reverse, surprising without context, and reflects a real trade-off.

All three current ADRs are **inherited from the sibling Bilbul project** (`2026 Juvenile gut evac`) — they are protocol-level decisions that travel site-agnostically. Site-specific differences (e.g. static-pond fast logistics, larger π uncertainty) are recorded in the Consequences sections, not as new ADRs.
