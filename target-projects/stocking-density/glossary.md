# Glossary — 2023 Stocking Density

Domain vocabulary for this project. Add new terms here when they first resolve.

**Stocking density** — Number of fish per unit cage volume at the start of a cycle. The single independent variable in this trial.

**Rectangular cage** — Cage geometry used throughout this trial. The cage-design choice is held constant; only stocking density varies. (See sibling `2021 Cages Round vs Rectangular` for the geometry trial.)

**FY23/24** — Australian financial year 2023–2024 (Jul 2023 – Jun 2024). The trial period.

**Erin Bubner / Trent D'Antignana** — Authors of the final report. Both are external scientific contributors (Nutrisea Pty Ltd).
